  1. Add the script in <head>:
    • Placing the script in the <head> blocks HTML parsing until the script is downloaded and executed. 
    
<!DOCTYPE html>
<html>
<head>
    <title>Script in Head</title>
    <script src="script.js"></script>
</head>
<body>
    <h1>Hello, JavaScript!</h1>
</body>
</html>
  2. Add the script at the end of the <body> tag:
    • Placing the script at the end of the <body> ensures the HTML is fully parsed before the script is executed. 
    
<!DOCTYPE html>
<html>
<head>
    <title>Script at End of Body</title>
</head>
<body>
    <h1>Hello, JavaScript!</h1>
    <script src="script.js"></script>
</body>
</html>
  3. Use the async attribute:
    • The async attribute allows the script to download asynchronously without blocking HTML parsing. The script executes as soon as it is downloaded, even if HTML parsing is incomplete. 
    
<!DOCTYPE html>
<html>
<head>
    <title>Async Script</title>
    <script src="script.js" async></script>
</head>
<body>
    <h1>Hello, JavaScript!</h1>
</body>
</html>
  4. Use the defer attribute:
    • The defer attribute allows the script to download asynchronously but executes only after HTML parsing is complete. 
    
<!DOCTYPE html>
<html>
<head>
    <title>Defer Script</title>
    <script src="script.js" defer></script>
</head>
<body>
    <h1>Hello, JavaScript!</h1>
</body>
</html>
  5. Understand the differences:
    • Script in <head>: Blocks HTML parsing until the script is executed.
    • Script at end of <body>: Executes after HTML parsing is complete.
    • async: Downloads and executes the script asynchronously, without blocking HTML parsing.
    • defer: Downloads the script asynchronously but executes it only after HTML parsing is complete.

Steps to Test:

  1. Create an HTML file for each method (e.g., head.html, body-end.html, async.html, defer.html).
  2. Add the corresponding script loading method to each file.
  3. Open each file in a browser and observe the behavior using the browser's DevTools (Network and Console tabs).
  4. Note the differences in script execution timing and its impact on page rendering.
