Task 2: Write Your First JavaScript Code

  1. Create a new file: index.html:
    • Open VS Code and create a new file named index.html.
  2. Add a <script> tag inside the HTML file or link an external script.js:
    • Option 1: Add inline JavaScript inside the <script> tag.
    • Option 2: Link an external JavaScript file using the src attribute.
  3. Inside script.js, write a simple greeting program: 
    
        // script.js
        console.log("Hello, tapaScript!");
        console.log("Welcome to 40 Days of JavaScript!");
        document.write("Check the browser console for a message!");
  4. Run the HTML file in the browser and check:
    • The console log prints the text.
    • The document.write() outputs text on the webpage.

Example Code:

Option 1: Inline JavaScript


        <!DOCTYPE html>
        <html>
        <head>
            <title>My First JavaScript</title>
        </head>
        <body>
            <h1>Hello, JavaScript!</h1>
            <script>
                console.log("Hello, tapaScript!");
                console.log("Welcome to 40 Days of JavaScript!");
                document.write("Check the browser console for a message!");
            </script>
        </body>
        </html>

Option 2: External JavaScript


        <!DOCTYPE html>
        <html>
        <head>
            <title>My First JavaScript</title>
        </head>
        <body>
            <h1>Hello, JavaScript!</h1>
            <script src="scripts.js"></script>
        </body>
        </html>

Steps to Run:

  1. Save the index.html and script.js files.
  2. Open index.html in a browser (double-click the file or use Live Server in VS Code).
  3. Open the browser console (Right-click → Inspect → Console) to see the logged messages.
  4. Check the webpage for the output of document.write().
