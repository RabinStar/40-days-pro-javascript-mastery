Task 1: Setting Up Your Environment

  1. Install VS Code (if not installed):
    • Download and install VS Code from code.visualstudio.com.
  2. Set up Live Server or open the HTML file directly in a browser:
    • Install the Live Server extension in VS Code.
    • Alternatively, open your HTML file directly in a browser by double-clicking the file.
  3. Open the DevTools Console:
    • Right-click on the webpage → Select Inspect → Go to the Console tab.
    • Alternatively, use the keyboard shortcut: Ctrl + Shift + I (Windows/Linux) or Cmd + Option + I (Mac).
